# DocumentLike

Запуск из главной директории:

	mkdir build && cd build
    cmake .. -DCMAKE_BUILD_TYPE=Release
    cmake --build .

------------------------------------------------------------------------------------
    ./DocumentLike <filename>

Например:

    ./DocumentLike document.dat
    
------------------------------------------------------------------------------------

Возможно понадобится установить библиотеку fmt:

    brew install fmt
